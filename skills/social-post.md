# Skill: Social Media Post Creation

## Purpose
Create platform-specific social media content for MarketWake.

## When to Use
- Content calendar requires social posts
- Repurposing blog content for social distribution
- Standalone social content creation
- Trend-reactive posts

## Procedure

1. **Identify platform**: LinkedIn, Instagram, X/Twitter, Facebook
2. **Select content pillar**: Education (40%), Authority (30%), Awareness (20%), Conversion (10%)
3. **Choose persona**: Who is this reaching?
4. **Generate 3 hook variants** for testing

5. **Write platform-specific content**:

### LinkedIn Format
```
[Hook line — stop the scroll]

[2-3 short paragraphs with insight/value]

[Key takeaway or bold statement]

[CTA or engagement question]

#hashtag1 #hashtag2 #hashtag3
```
- Aim for 500-1,200 characters
- Personal POV from MW team member when possible
- Include line breaks for readability

### Instagram Format
```
Caption:
[Hook] + [Value/Story] + [CTA]

Hashtags: [5-15 relevant tags]

Carousel Slides (if applicable):
Slide 1: [Hook/Title]
Slide 2-8: [One idea per slide]
Slide 9-10: [Summary + CTA]
```

### X/Twitter Format
```
Single Tweet: [Under 280 chars — hook + value + link]

Thread:
1/ [Hook tweet — must stand alone]
2-6/ [Value tweets — one point each]
7/ [CTA tweet with link]
```

### Facebook Format
```
[Question or hook to prompt engagement]

[Story or insight — can be longer form, up to 500 words]

[CTA or discussion prompt]
```

6. **Deliver with metadata**:
```
Platform:
Pillar:
Persona:
Hook Variants: [3 options]
Post Copy:
Hashtags:
Suggested Visual: [description of image/graphic needed]
Best Time to Post: [recommendation]
```
