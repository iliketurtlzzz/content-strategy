# Skill: SEO Content Brief

## Purpose
Generate a comprehensive SEO brief that guides content creation for search-optimized pages.

## When to Use
- Before writing any blog post targeting organic search
- Service page optimization
- Content refresh/update planning

## Procedure

1. **Keyword research inputs** (ask user or research):
   - Primary keyword + search volume estimate
   - Secondary keywords (3-5)
   - Search intent (informational, commercial, transactional, navigational)
   - Current ranking position (if updating existing content)

2. **Competitive analysis**:
   - What's ranking in the top 5 for this keyword?
   - What angles are they covering?
   - What's missing that MW can uniquely provide?
   - Average word count of top results

3. **Build the brief**:

```
# SEO Content Brief

## Target Keyword
- Primary: [keyword] (est. volume: [X]/mo)
- Secondary: [keyword list]
- Search Intent: [informational/commercial/transactional]

## Content Specifications
- Format: [blog post / guide / landing page]
- Word Count: [range based on competitive analysis]
- Target URL: [slug]
- Target Persona: [from profiles/]
- Content Pillar: [Education/Authority/Awareness/Conversion]

## Search Intent Analysis
- What the searcher wants: [describe]
- What they should do after reading: [desired action]

## Required Coverage
Topics/questions this content MUST address:
1. [Topic/question from SERP analysis]
2. [Topic/question]
3. [Topic/question]
...

## Recommended Structure
- H1: [suggested title with primary keyword]
- H2s: [suggested section headings]
- H3s: [subsections if needed]

## Internal Linking Targets
- Link TO: [relevant MW pages]
- Link FROM: [existing MW pages that should link to this]

## Competitive Gaps (MW's Angle)
- Competitors miss: [gap]
- MW unique perspective: [angle]

## On-Page SEO Checklist
- [ ] Primary keyword in title, H1, first 100 words, meta description
- [ ] Secondary keywords distributed naturally
- [ ] Meta description: 150-160 chars with keyword + CTA
- [ ] Image alt text with keyword
- [ ] Internal links (2-3 minimum)
- [ ] External links to authoritative sources
- [ ] URL slug: short, keyword-rich
```

## Output
Deliver the complete brief so a writer (human or AI) can produce optimized content without additional research.
